# Tournament-Tracker

### Webtech Projekt von Dennis Gross

## Beschreibung
Der Tournament-Tracker soll Turniere Dokumentieren und den Tunierverlauf visualisieren. 

Je nach Anzahl der Spieler, soll ein Tunierbaum automatisch generiert werden.
